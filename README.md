# Airbnb Power BI Dashboard

## Project Overview

This interactive Power BI dashboard explores Airbnb listings from around the world, providing insights into pricing, guest ratings, host activity, and geographical distribution.

Designed for storytelling, analysis, and a clean visual experience, this dashboard is ideal for portfolio presentation and business reporting.

---

## Key Features

- Analyzed Airbnb listings from **119+ countries**
- Displayed **average, minimum, and maximum prices**
- Built a **scatter plot** to analyze the relationship between price and guest ratings
- Created a **donut chart** showing listing distribution by country
- Added interactive filters for:
  - Country
  - Guest Rating
  - Host Name
- Highlighted top high-priced Airbnb listings
- Implemented dynamic slicers and interactive visuals
